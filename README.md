# Sports Data Analytics Portfolio

##  Project Overview

This repository showcases end-to-end data analysis and machine learning projects on sports datasets, including **FIFA 22 player data** and a **sports training performance dataset**.

The goal of these projects is to extract meaningful insights, understand performance patterns, and build predictive models to support data-driven decision-making in sports analytics.

## Datasets Used

### FIFA 22 Player Dataset

* Player attributes (age, overall rating, potential)
* Market value and wage data
* Club and nationality information

###  Sports Training Dataset

* Session duration
* Heart rate and speed metrics
* Distance covered
* Endurance and technique scores
* Performance level classification

##  Key Analysis & Insights

###  FIFA 22 Analysis

* Analyzed relationship between **player value and overall rating**
* Identified **top-performing players** and **emerging talents**
* Evaluated **age vs performance trends**
* Compared **player performance across positions and countries**
* Built a **Random Forest model** to predict player market value

---

### Sports Training Analysis

* Analyzed athlete performance based on training metrics
* Explored relationships between:

  * Heart rate
  * Speed
  * Endurance
  * Performance levels
* Built a **classification model** to predict performance level
* Identified key factors influencing athletic performance

---

## 🤖 Machine Learning Models

### FIFA 22

* **Model:** Random Forest Regressor
* **Goal:** Predict player market value
* **Features Used:** Age, Overall Rating, Potential

### Sports Training Dataset

* **Model:** Random Forest Classifier
* **Goal:** Predict performance level
* **Features Used:**

  * Session Duration
  * Heart Rate Avg
  * Speed Avg
  * Distance Covered
  * Endurance Score
  * Technique Score
